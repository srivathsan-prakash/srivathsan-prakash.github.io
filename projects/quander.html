Quander


Engine: Unity

Platforms: WebGL

Links: Game Info | Link to Play


A game made in collaboration with The University of Chicago featuring a collection of minigames teaching Quantum Computing concepts to middle schoolers

  • Build the overarching game-selection hub containing shared functionality
    • Organize, guide, and assist 4 student devs through meetings, documentation, and coding support; while also collaborating with our own multidisciplinary team of producers, artists, UX designers, and game designer
    • Create an event system following the observer pattern; allowing students to call shared functionality with simple one-liners while also decoupling and simplifying complex systems in the game
  • Implement 1 out of 5 minigames (Buried Treasure)
  • Utilize a data-driven approach and set up a cloud based system to give the client full control of content and related features
